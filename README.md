# DM Holidays

## Project Structure
```
CSS/
├── client/   → React + Vite frontend
└── server/   → Express + MongoDB backend
```

## Start Backend
```
cd server
npm run dev
```
Runs on: http://localhost:5000

## Start Frontend
```
cd client
npm install
npm run dev
```
Runs on: http://localhost:5173

## Admin Panel
Visit: http://localhost:5173/admin
Password: set in server/.env → ADMIN_PASSWORD
